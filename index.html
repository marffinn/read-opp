PDF CRM Opportunity Extractor

Web Interface Instructions

This is a static HTML page that provides instructions for using the PDF CRM Extractor web interface.

How to Use the Web Interface

  1. Open a terminal or command prompt
  2. Navigate to your project directory:
    cd path/to/read-opp
  3. Run the web interface:
    python simple_app.py
  4. Open your browser and go to:
    http://127.0.0.1:5000/
  5. Upload a PDF file and click "Upload & Extract"
  6. View the extracted CRM data and export it as JSON or CSV

About the PDF CRM Extractor

This tool extracts structured CRM opportunity data from PDF documents using a local LLM (Mistral 7B).

The extracted data includes:

  • Company name
  • Contact information
  • Opportunity value
  • Timeline
  • Products/services of interest
  • And more...
